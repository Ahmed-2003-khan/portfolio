This project focuses on building an intelligent and reliable system for predicting housing prices across various cities in Pakistan. The tool is designed to support both home buyers and real estate investors by providing data-driven property value estimations through an interactive web application.

üîç Problem Statement & Initial Challenges

The project began with the goal of collecting real estate data directly from Zameen.com, a popular Pakistani property portal. However, the web-scraped dataset lacked many important features, leading to a modest 83% accuracy in predictions.

To overcome this limitation, I transitioned to a more comprehensive and feature-rich Kaggle dataset, which significantly improved the model's performance by offering deeper insights into the Pakistani real estate landscape.

ü§ñ Model Development & Optimization

  • Evaluated multiple regression models including Linear Regression, Random Forest, and Gradient Boosting. XGBoost emerged as the most effective model.
  • Performed hyperparameter tuning using GridSearchCV to maximize model performance.
  • Built two separate models: Sale Model (R¬≤ = 98%) and Rental Model (R¬≤ = 89%).
  • Utilized Mahalanobis distance for outlier detection to maintain data integrity and enhance prediction accuracy.
  • Models accurately capture the trends and variables influencing housing prices in different regions of Pakistan.

üìä Feature Engineering & Data Processing

  • Outlier detection and removal.
  • Handling of missing values and noisy data.
  • Encoding of categorical features like city, property type, and number of bedrooms.
  • Feature scaling to normalize property size and price ranges.

The thoughtful feature engineering process was crucial in improving model generalization and reducing overfitting.

üåê Web Application Deployment

  • Developed a fully interactive web application using Streamlit, hosted on the Streamlit Community Cloud.
