🚔 Crimes Data Analysis – Los Angeles, California

This project focused on analyzing crime patterns in Los Angeles using Python, with the goal of uncovering insights that could support public safety efforts.

🔍 Tools & Technologies

  • Python (Jupyter Notebook)
  • Matplotlib & Seaborn for data visualization and trend analysis

📊 Key Work

  • Cleaned and explored a large dataset of reported crimes in Los Angeles.
  • Analyzed trends by time, location, and type of crime.
  • Created visualizations to highlight:
    • Crime hotspots across neighborhoods
    • Peak crime hours and seasonal patterns
    • Most frequent crime categories

🧠 Outcome

The analysis revealed key trends and patterns that can inform law enforcement strategies and public awareness. This project strengthened my skills in data exploration, visualization, and communicating insights effectively.
